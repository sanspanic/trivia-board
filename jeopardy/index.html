JavaScript Workflow

  • Restart button triggers request to API
  • 6 random categories are picked using Lodash, their IDs are used to obtain 5 random clues
  • Table is generated via DOM Manipulation with jQuery using nested loops. Clue and category IDs are retained in cell ID, so they can be used to access corresponding clue
  • Event listener added to cells which displays clue and answer respectively upon click
